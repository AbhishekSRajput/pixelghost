# PixelGhost Support

Get help with PixelGhost, report a bug, or suggest an improvement through
GitHub Issues:

- [Search existing issues](https://github.com/AbhishekSRajput/pixelghost/issues)
- [Open a new issue](https://github.com/AbhishekSRajput/pixelghost/issues/new)

You can also contact support at <codecube99@gmail.com>.

## Before opening an issue

PixelGhost works on ordinary `http://` and `https://` pages. Browser-internal
pages, extension stores, and other restricted pages do not allow extensions
to inject overlays. If the affected page is supported, reload it and try the
action again before reporting the problem.

## What to include

- The PixelGhost version and Chrome version.
- Clear steps to reproduce the problem.
- What you expected and what happened instead.
- Whether the problem affects overlays, the popup, the floating widget, or
  saved projects.
- Screenshots or error messages, if they are safe to share.

## Privacy and security

GitHub issues are public. Do not post private overlay images, sensitive page
URLs, personal information, or other confidential data. Send privacy requests,
security reports, or support that requires private details to
<codecube99@gmail.com>.

See the [privacy policy](PRIVACY.md) for details about how PixelGhost handles
data.
